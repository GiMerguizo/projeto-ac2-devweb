# Projeto AC2
Entrega do Projeto da AC2 da Faculdade com Front-end e Back-end

## Parte 1 - Frontend
### Ferramentas
- Angular
- Html
- CSS
- Bootstrap

## Parte 2 - Backend
- NodeJS
- Express
- Mongo Atlas

### Comandos
- `npm install express`
- `npm install mongoose`
- `npm install jsonwebtoken`
- `npm install bcryptjs`
- `npm install dotenv`
<br>
Iniciar projeto: `npm run dev`
